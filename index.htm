Important Formulas

Sets and Number Systems

Union of Sets: A ∪ B = {x | x ∈ A or x ∈ B}

سیٹوں کا یونین: A ∪ B = {x | x ∈ A یا x ∈ B}

Intersection of Sets: A ∩ B = {x | x ∈ A and x ∈ B}

سیٹوں کا انٹرسیکشن: A ∩ B = {x | x ∈ A اور x ∈ B}

Functions and Quadratics

Quadratic Formula: x = [-b ± √(b² - 4ac)] / 2a

کویڈریٹک فارمولا: x = [-b ± √(b² - 4ac)] / 2a

Discriminant: D = b² - 4ac

ڈسکریمیننٹ: D = b² - 4ac

Matrices and Determinants

2x2 Determinant: det [a b; c d] = ad - bc

2x2 میٹرکس کا ڈیٹرمننٹ: det [a b; c d] = ad - bc

Sequences and Series

Arithmetic Sequence: aₙ = a₁ + (n-1)d

اریتھمیٹک سیکوئنس: aₙ = a₁ + (n-1)d

Geometric Sequence: aₙ = a₁ × rⁿ⁻¹

جیومیٹرک سیکوئنس: aₙ = a₁ × rⁿ⁻¹
